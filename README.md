# Student Social Media Addiction Risk Predictor

## Project Overview
The Student Social Media Addiction Risk Predictor is a Machine Learning project that analyzes students' social media usage patterns, sleep habits, mental well-being, and social interactions to predict their level of social media addiction risk.

## Problem Statement
The increasing use of social media among students has led to concerns about addiction, reduced academic performance, poor sleep quality, mental health issues, and social conflicts. Identifying students who are at risk of developing social media addiction is challenging due to the complex relationship between behavioral, academic, and psychological factors. This project aims to develop a Machine Learning-based solution that analyzes student usage patterns and well-being indicators to predict social media addiction risk levels, enabling early awareness and informed decision-making.

## Dataset Information
The dataset contains information about 705 students and 13 attributes related to their social media usage behavior, academic impact, mental well-being, and lifestyle habits. It includes demographic details, daily social media usage patterns, sleep duration, mental health scores, relationship status, and the effects of social media on academic performance. The dataset was used to analyze behavioral trends, perform feature engineering, and build a machine learning model for predicting students' social media addiction risk levels.

Dataset Size: 705 records × 13 columns

## 📦 Installation (Local Setup)
1. Clone the repository:
git clone <repository-url>
cd Student-Social-Media-Addiction-Prediction
2. Create a virtual environment:
python -m venv venv
3. Activate the virtual environment:
**Windows**
venv\Scripts\activate
**Linux/Mac**
source venv/bin/activate
4. Install the required dependencies:
pip install -r requirements.txt
5. Run the Streamlit application:
streamlit run app.py
6. Open the application in your browser using the URL displayed in the terminal (typically `http://localhost:8501`).
